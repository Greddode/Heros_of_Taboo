# Heroes of Taboo — Build Guide

## Prerequisites

### Windows (w64devkit)

1. Install [w64devkit](https://github.com/skeeto/w64devkit/releases) to `C:\raylib\w64devkit\`
2. Verify:
   ```cmd
   C:\raylib\w64devkit\bin\gcc.exe --version
   C:\raylib\w64devkit\bin\make.exe --version
   ```
3. Build raylib into `bin/Debug/raylib.lib` (see below)

### macOS / Linux

```sh
# macOS
brew install premake5

# Ubuntu/Debian
sudo apt-get install premake5 libgl1-mesa-dev libxrandr-dev libxinerama-dev \
  libxcursor-dev libxi-dev libxkbcommon-dev
```

## Building Raylib

From the repo root:
```sh
git clone --depth 1 --branch 5.5 https://github.com/raysan5/raylib.git raylib-master

# Windows (w64devkit)
cd raylib-master\src
C:\raylib\w64devkit\bin\make.exe PLATFORM=PLATFORM_DESKTOP
mkdir ..\..\bin\Debug
copy libraylib.a ..\..\bin\Debug\raylib.lib

# Linux/macOS
cd raylib-master/src
make PLATFORM=PLATFORM_DESKTOP -j$(nproc)
mkdir -p ../../bin/Debug
cp libraylib.a ../../bin/Debug/
```

## Compile

```sh
cd game
make config=debug_x64          # debug build
make config=release_x64        # optimized build
```

Windows: use the full paths:
```cmd
cd game
C:\raylib\w64devkit\bin\make.exe config=debug_x64
```

## Run Tests

```sh
cd game
make config=debug_x64 test
```

Expected: 28 tests pass in under 2 seconds.

## Clean

```sh
make clean    # removes obj/ and bin/Debug/*.exe
```

## Project Structure

```
bin/          # compiled output (ignored by git)
  Debug/      # Heroes of Taboo.exe, test_runner.exe, raylib.lib
  Release/    # Heroes of Taboo.exe, raylib.lib
game/
  Makefile    # build targets (generated by premake5)
  src/        # all game source files
  include/    # shared public headers
engine/       # ECS core + resource manager
raylib-master/ # raylib source (gitignored)
```

## Troubleshooting

**`raylib.lib` not found:** Build raylib first (see above). The Makefile expects it at `bin/Debug/raylib.lib`.

**`gcc` / `make` not found:** Ensure w64devkit is at `C:\raylib\w64devkit\` and its `bin/` folder is on `PATH`, or use the full path.

**Compile errors about missing includes:** Ensure raylib-master is cloned at the repo root. The Makefile includes `-I../raylib-master/src`.

**Test runner crashes:** Run `make clean` then `make config=debug_x64 test` to force a full rebuild.
